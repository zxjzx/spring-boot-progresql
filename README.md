项目简介：
实现了增删查改；发送邮件

这是一个javaweb，使用的框架是springboot，mybatis,后台数据库使用的是postgresql(和mysql很相似)

jdk8,下载maven配置后

run SpringBootProgresqlApplication.java即可启动项目
